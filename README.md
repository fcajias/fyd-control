# FYD Control de Equipos Informáticos

Sistema web para el control y seguimiento de salida de equipos (laptops) para teletrabajo.

## Roles del sistema
- **Líder del Área** — Registra asesores y autoriza las salidas diarias
- **Sistemas (TI)** — Monitorea equipos, registra salidas/ingresos e incidencias *(clave de acceso requerida)*
- **Seguridad** — Visualiza la lista del día e imprime la hoja de control

## Tecnología
- HTML + CSS + JavaScript puro (sin dependencias externas)
- Almacenamiento local con `localStorage` (versión offline)
- Próximamente: integración con Supabase (base de datos en la nube)

## Uso
Abre `index.html` directamente en el navegador o accede vía GitHub Pages.
